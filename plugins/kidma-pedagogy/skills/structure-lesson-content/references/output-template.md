# Kidma Lesson Content Extraction — Output Template

This is the exact structure every extracted lesson must follow. Every section is required. If information is not found in the source materials, write "Not found in source materials" — never skip a section or invent content.

The file is split into two parts:
- **Part A: English** — The full extracted content translated to English. This is the primary readable content.
- **Part B: Hebrew Source** — The original Hebrew text, organized by the same section numbers for cross-reference.
