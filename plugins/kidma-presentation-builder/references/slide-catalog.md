## Slides 3–12 — Basic Content (10 Color Variations)

All share the same text layout. Only the decorative circles differ.

### Common Layout
- **Right-aligned text blocks:**
  - Title (כותרת) — large, bold
  - Description (תיאור) — medium, lighter
  - Content block (תוכן) — body text area
  - 3 bullet lines (משפט אחד / שני / שלישי)
- **Decorative element:** Colored circle(s) in corners

### Color Variations

| Slide | Top-Left Circle | Bottom-Right Circle | Best For |
|-------|----------------|---------------------|----------|
| 3 | None | None | Plain content, when you don't want visual distraction |
| 4 | Blue (large) | None | First concept in a sequence |
| 5 | Green (large) | None | Second concept in a sequence |
| 6 | Pink (large) | None | Third concept in a sequence |
| 7 | Blue (large) | Green (small) | Concept with dual emphasis |
| 8 | Pink (large) | Green (small) | Concept with dual emphasis |
| 9 | Pink (large) | Blue (small) | Concept with dual emphasis |
| 10 | Blue (overflow corner) | Green (small) | Content with visual framing |
| 11 | Pink (overflow corner) | Green (corner) | Content with visual framing |
| 12 | Pink (overflow corner) | Green (corner) | Content with visual framing |

### Placeholder Replacement Guide
```
כותרת  → Section/concept title (e.g., "פרומפט — Prompt")
תיאור  → Short explanation (e.g., "ההוראה שאנחנו נותנים לבינה המלאכותית")
תוכן   → Content label (e.g., "דוגמאות:")
משפט אחד → First point
משפט שני  → Second point
משפט שלישי → Third point
```

**Pedagogical mapping:** הבנה phase — one key concept per slide
