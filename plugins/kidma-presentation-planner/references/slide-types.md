# Kidma Template Slide Types — Quick Reference

> Use this to select the right slide type during presentation planning. For full technical details (placeholders, XML, colors), see the builder skill's slide-catalog.md.

## Available Slide Types

### Cover (Slide 2)
- **Use for:** Opening slide — always first
- **Content:** Catchy title, topic subtitle, lesson number, one-line description
- **Visual:** Dreamy cloud background, large centered text
- **Capacity:** 4 text fields

### Topics Overview — Dots (Slide 13)
- **Use for:** Lesson agenda / "מה נלמד היום?"
- **Content:** 4 topic names (short, 2-4 words each)
- **Visual:** 4 colored dots (green, pink, blue, gold) with topic labels
- **Capacity:** 4 topic names, no descriptions

### Topics Overview — Bars with Descriptions (Slide 14)
- **Use for:** Vocabulary/definitions overview, or agenda with more detail
- **Content:** 4 terms/topics with short descriptions + optional tip box
- **Visual:** Colored vertical bars, optional 💡 tip callout
- **Capacity:** 4 titles + 4 descriptions + 1 tip

### Basic Content (Slides 3–12) — 10 color variations
- **Use for:** Any single concept, definition, explanation, example
- **Content:** Title + description + content label + 3 bullet points
- **Visual:** White background with decorative colored circles
- **Color cycle:** Blue (4) → Green (5) → Pink (6) → then mixed (7-12)
- **Rule:** Never use the same color consecutively
- **Capacity:** 1 title + 1 description + 3 bullets

### Quote + Equation (Slide 15)
- **Use for:** A formula or combination concept (e.g., "role + context + constraints = great prompt")
- **Content:** Quote with attribution + 3 components + combined result
- **Visual:** Gray quote band + 3 colored blocks with "+" signs + "=" result
- **Capacity:** 1 quote + 3 components with descriptions + 1 conclusion

### Pyramid (Slide 16)
- **Use for:** Hierarchical concepts (e.g., AI → ML → Deep Learning → GenAI)
- **Content:** 3 layers with names and descriptions
- **Visual:** 3-tier pyramid (pink top, blue middle, green base)
- **Capacity:** 1 title + 3 layers (name + description each)

### Icon Rows + Highlight (Slide 17)
- **Use for:** 3 rules, principles, or key points with a summary
- **Content:** 3 principles with emoji icons + bottom highlight bar
- **Visual:** Colored square icons (📖💬🤖) + cream highlight bar
- **Capacity:** 1 title + 3 principles + 1 bottom summary

### Three Component Cards (Slide 18)
- **Use for:** Breaking a concept into 3 equal parts
- **Content:** 3 cards with emoji + title + description + summary sentence
- **Visual:** Side-by-side cards with colored borders (green/pink/blue)
- **Capacity:** 1 title + 3 cards (emoji + title + description) + 1 summary

### Three-Step Process (Slide 19)
- **Use for:** Sequential workflows, step-by-step processes
- **Content:** 3 steps with numbers, titles, and descriptions
- **Visual:** RTL flow — Blue 1️⃣ → Pink 2️⃣ → Green 3️⃣ with arrows
- **Capacity:** 1 title + 3 steps (title + description) + 1 summary

### Problem / Solution (Slide 20)
- **Use for:** Good vs bad comparison, before/after, prompt comparison
- **Content:** Problem example + solution example + prompt/text sample + tip
- **Visual:** Red-bordered ❌ box + green-bordered ✅ box + blue tip bar
- **Capacity:** 1 title + 1 problem + 1 solution + 1 prompt sample + 1 tip

### Numbered Task with Tool Badges (Slide 21)
- **Use for:** Creation phase assignment — the main student task
- **Content:** 4 numbered steps with specific AI tool for each step + mission statement
- **Visual:** Numbered rows with colored tool badges (tool names)
- **Capacity:** 1 title + 4 tasks (title + description + tool name) + 1 mission
- **Rule:** Tool names must be specific (e.g., "Gemini", "DALL·E", "Canva")

### Summary Checklist (Slide 22)
- **Use for:** Closing slide — always last
- **Content:** 4 key takeaways + closing discussion question
- **Visual:** Checkmarks ✓ + green highlight bar with 🤔 question
- **Capacity:** 1 title + 1 subtitle + 4 takeaways + 1 question

## Slide Selection Guide

| Content Type | Best Slide Type | Alternative |
|---|---|---|
| Lesson opening | Cover (2) | — |
| Agenda / topics list | Dots (13) | Bars (14) if need descriptions |
| Single concept/definition | Content (3-12) | Bars (14) for multiple terms |
| Vocabulary list | Bars (14) | Content (3-12) one term per slide |
| Hierarchy | Pyramid (16) | — |
| 3 rules/principles | Icon Rows (17) | Cards (18) |
| 3 components | Cards (18) | Icon Rows (17) |
| Step-by-step process | Process (19) | Content slides in sequence |
| Formula / combination | Quote+Equation (15) | — |
| Good vs bad example | Problem/Solution (20) | — |
| Student task | Task (21) | — |
| Lesson closing | Summary (22) | — |

## Typical Deck Structure (10-15 slides)

```
1. Cover (2)           — הקדמה
2. Agenda (13/14)      — סקירה
3-7. Content (3-12)    — הבנה (concepts, vocabulary, process)
8. Comparison (20)     — התנסות (good vs bad example)
9-10. Content (3-12)   — התנסות (guided task context)
11. Task (21)          — יצירה (creation assignment)
12. Summary (22)       — סיכום
```

Adjust based on lesson complexity — simpler lessons may need only 10 slides, concept-heavy lessons may need 14-15.

---

## Rules for Slide Sequencing

1. **Always start** with Cover (Slide 2)
2. **Always end** with Summary (Slide 22)
3. **Never repeat** the same slide layout consecutively
4. **Cycle colors** for content slides (blue→green→pink→orange)
5. **One concept per content slide** — don't cram
6. **Task slide (21) comes near the end** — before summary
7. **Problem/Solution (20) works best** in the middle — after concepts, before task
